# Email Validator
## Project Overview
This Email Validator application is a simple tool designed to verify 
whether a provided email address is valid based on format and, optionally, its domain existence. 
The app checks for valid email structure, such as the presence of an "@" symbol and a valid domain, 
and ensures that the format follows the standard rules of email addresses.

## Features
1. Basic Validation: Verifies the basic format of an email (e.g., name@example.com).
2. Domain Check: checks if the domain (e.g., example.com) exists (via DNS lookup).
3. Real-time Validation: Validates the email as the user types (for web-based applications).
4. Regex Validation: Uses Regular Expressions (Regex) to ensure emails adhere to the correct format.

## Technology Stack
1. Front-end: HTML5, CSS3, JavaScript
2. Back-end : Node.js, Express.js
